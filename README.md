# UpSight

AI-powered Chrome extension for Upwork freelancers to analyze jobs, screen opportunities, and generate proposals efficiently.

## Problem

As Upwork freelancers, we spend hours every day:
- Scrolling through job listings to find the right opportunities
- Manually checking client history and credibility
- Writing personalized proposals for each application
- Wondering if a job is worth our precious Connects

## Solution

UpSight uses AI to automate the job screening process, helping you:

- **Analyze Jobs** - Get instant match scores based on your profile
- **Assess Clients** - Evaluate client credibility and risk before applying
- **Screen in Bulk** - See recommendation scores directly on the job list
- **Generate Proposals** - Create professional proposal drafts in minutes

## Features

- AI-powered job analysis with match scoring
- Client risk assessment and history evaluation
- Batch pre-screening on job listings
- Proposal generation assistance
- Profile sync from Upwork
